# 🚀 My Hearst AI - Claude CI/CD Cockpit

Production-ready Claude AI project management system.

## Features

- ✅ Project management
- ✅ Job execution with Claude API
- ✅ Version control
- ✅ Dark/Light mode
- ✅ PDF export
- ✅ Notifications system

## Stack

- Frontend: HTML/CSS/JS (Vanilla)
- Backend: Node.js + Express
- Database: PostgreSQL (Supabase)
- Deployment: Vercel
- CI/CD: GitHub Actions
