# 📍 Où trouver le bouton "Storage" dans Vercel

## 🎯 Chemin exact pour accéder à Storage

### Méthode 1 : Via le menu de gauche du projet

1. **Allez sur votre projet** :
   - https://vercel.com/adrien-nejkovics-projects/hearst-ai-v3
   - Ou cliquez sur le projet depuis le dashboard

2. **Dans le menu de gauche**, vous verrez plusieurs sections :
   ```
   📊 Overview
   🚀 Deployments
   ⚙️  Settings
   💾 Storage  ← CLIQUEZ ICI
   ```

3. **Cliquez sur "Storage"** dans le menu de gauche

4. **Sur la page Storage**, vous verrez :
   - En haut à droite : bouton **"Create Database"** ou **"Add Database"**
   - OU au centre : une carte avec **"Create Database"**

---

### Méthode 2 : Via Settings

1. Allez sur votre projet
2. Cliquez sur **"Settings"** dans le menu de gauche
3. Dans Settings, cherchez la section **"Storage"** ou **"Databases"**
4. Cliquez dessus

---

### Méthode 3 : URL directe

**URL directe vers Storage** :
```
https://vercel.com/adrien-nejkovics-projects/hearst-ai-v3/storage
```

## 🔍 Si vous ne voyez pas "Storage"

1. **Vérifiez que vous êtes sur le bon projet** : `hearst-ai-v3`
2. **Vérifiez votre plan Vercel** : Storage est disponible sur tous les plans
3. **Essayez de rafraîchir la page** (F5 ou Cmd+R)
4. **Vérifiez que vous êtes connecté** à votre compte Vercel

---

## 🎯 Une fois sur la page Storage

Vous verrez :
- **"Create Database"** en haut à droite (bouton bleu/vert)
- OU une grande carte au centre avec **"Create Database"**

Cliquez dessus → Sélectionnez **"Postgres"** → Créez votre base !

---

**Le dashboard est ouvert - cherchez "Storage" dans le menu de gauche !**
