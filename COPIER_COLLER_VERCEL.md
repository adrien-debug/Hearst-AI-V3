## 📝 Instructions rapides

1. **Ouvrez Vercel Dashboard**
2. **Votre projet** → **Settings** → **Environment Variables**
3. **Cliquez sur "Add New"**
4. **Copiez-collez** chaque valeur ci-dessus
5. **Cochez** les environnements appropriés
6. **Cliquez sur "Save"**

---

## ✅ Après avoir ajouté toutes les variables

1. **Redéployez** :
   - Dashboard → Deployments → Redeploy
   - Ou : `vercel --prod`

2. **Vérifiez** :
   - L'application devrait fonctionner sans erreurs 500
   - Les routes API devraient répondre correctement
